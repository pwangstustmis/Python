Recipe 4：程式邏輯- 條件

本單元將說明 if命令、if else命令、if elif else命令、if命令簡單表示、多條件(and、or、not)

if命令

當判斷條件只有對的時後才執行時，Python使用 if 命令並設定條件對時動作，設定方式如下，if命令最後須使用:來定義新的程式區塊，新的程式區塊必須縮排，請注意：新的程式區塊必須縮排，否則會發生錯誤。

if (2>1): print('Yes') #錯誤寫法 if (2>1): print('Yes')
練習：請利用input()方法輸入兩個空白隔開的數，印出這兩個數中較大的數

if else命令

當判斷條件有對與錯時執行動作，Python使用 if else 命令並設定條件對與錯動作，設定方式如下，if、else命令最後須使用:來定義新的程式區塊，新的程式區塊必須縮排，請注意：新的程式區塊必須縮排，否則會發生錯誤。

if (2>1): print('Yes') else: print('No')

文字類型操作

文字在Python以萬國碼(Unicode)方式儲存，儲存方式類似清單，因此可利用字元位置取得字串內容，同時支援 in 運算，請注意字串長度可利用len函數為 len(字串或字串變數)，文字類型常用操作如下：

  • 切割(slicing)：切割用於取得文字部分內容，切割方式包含 [:]、[開始位置:]、[:結束位置]、[開始位置:結束位置]、[開始位置:結束位置:間距值]，分別代表 全部、開始位置之後、從頭到結束位置、開始到結束位置、開始到結束位置有間距。
    • 練習：請設定變數 school 值為 南臺科技大學，請利用文字切割印出 南臺科技大學、科技大學、南臺科技、臺科技、南科大。
  • upper()：用於將字串中英文字轉為大寫。
    • print('abc'.upper())
  • lower()：用於將字串中英文字轉為小寫。
    • print('ABC'.lower())
  • strip()：用於將字串中頭尾空白移除。
    • print(' ABC '.strip())
  • replace(原始字串,新字串,取代次數)：用於將字串中原始字串用新字串取代，如果未設定取代次數，預設為-1，代表全部取代。
    • print('How are you?'.replace('o','k'))
  • split(分割符號,分割次數)：用於將字串中依照分割符號進行分割，分割後將傳回字串清單，分割次數預設為-1，代表全部分割。
    • print('How are you?'.split(' '))
  • isnumeric()：用於判斷是否為數字性文字字串，傳回值為 True或False。
    • print('123'.isnumeric()) print('123a'.isnumeric())
  • find(子字串,開始位置,結束位置)：傳回字串中子字串第一次出現位置，開始位置用於設定字串搜尋開始位置，預設為0，結束位置用於設定字串搜尋結束位置，預設為字串長度，如果找不到則傳回值為 -1。
    • print('How are yor?'.find('How')) print('How are yor?'.find('how'))

練習：輸入一串文字，印出這串文字最前面兩個字與最後面兩個字合併，例如輸入南臺科技大學，則輸出南臺科學 練習：輸入一串文字，並反向印出這串文字，例如輸入南臺科技大學，則輸出學大技科臺南 練習：輸入一串文字，將這串文字中與第一個字相同，全部取代成$，例如輸入this is a cat，則輸出$his is a ca$ 練習：輸入三個文字以逗點隔開，第一個代表原始文字、第二個代表想要被取代文字、第三個代表取代文字，請印出取代前與取代後答案，例如輸入this is a cat. that is a dog.,is,IS，答案印出 this is a cat. that is a dog. this IS a cat. that IS a dog. 練習：輸入兩個文字以逗點隔開，第一個代表原始文字、第二個代表想要被移除文字，請印出取代前與取代後答案，例如輸入this is a cat. that is a dog.,is，答案印出 this is a cat. that is a dog. th a cat. that a dog. 練習：輸入兩個文字以逗點隔開，第一個代表原始文字、第二個代表想要被移除文字位置，請印出移除前與移除後答案，例如輸入this is a cat.,2，答案為置，請印出移除前與移除後答案，例如輸入tis is a cat.

清單類型操作

清單類型常用操作如下：

  • 切割(slicing)：切割用於取得部分清單內容，切割方式包含 [:]、[開始位置:]、[:結束位置]、[開始位置:結束位置]、[開始位置:結束位置:間距值]，分別代表 全部、開始位置之後、從頭到結束位置、開始到結束位置、開始到結束位置有間距。
    • 練習：請設定清單變數 x 值為 [1,2,3,4,5,6]，請利用切割印出 [1,2,3,4,5,6]、[3,4,5,6]、[1,2,3,4]、[2,3,4]、[1,3,5]。
  • append(清單值)：將新清單值新增於清單最後。
    • x=[1,2,3,4,5] x.append(6)
  • insert(位置,清單值)：將新清單值新增於清單特定位置。
    • x=[1,2,3,4,5] x.insert(0,6)
  • 刪除清單值：可利用 remove(清單值)、pop()、pop(位置值)、del 清單變數[位置值]、del 清單變數、清單變數.clear() 來刪除資料，分別代表 刪除清單值、刪除最後一個、刪除位置值清單項目、刪除位置值清單項目、移除整個清單變數、刪除清單變數內容(清單變數仍然存在)。
    • 練習：請利用range建立從1到10數，並利用list進行類別轉換後，練習刪除清單函數。
  • 清單排序：可利用 清單變數.sort()、清單變數.sort(reverse=True) 來進行 清單遞增(由小到大)或遞減(由大到小) 排序。
    • x=[5,4,3,2,1] x.sort() print(x) x.sort(reverse=True) print(x)
  • 清單複製：Python清單變數設定並不會產生新的請清單變數，請利用清單變數.copy() 來進行清單複製，請不要用 新變數=原始清單變數 的方式進行清單複製，因為這種方式並不會將清單複製到新變數，而是將兩個變數只到相同的清單。
    • x=[1,2,3,4,5] y=x y[0]=100 print(x) x=[1,2,3,4,5] z=x.copy() y[0]=100 print(x) print(y)
