Recipe 4：程式邏輯- 條件

本單元將說明 if命令、if else命令、if elif else命令、if命令簡單表示、多條件(and、or、not)

if命令

當判斷條件只有對的時後才執行時，Python使用 if 命令並設定條件對時動作，設定方式如下，if命令最後須使用:來定義新的程式區塊，新的程式區塊必須縮排，請注意：新的程式區塊必須縮排，否則會發生錯誤。

if (2>1): print('Yes') #錯誤寫法 if (2>1): print('Yes')
練習：請利用input()方法輸入兩個空白隔開的數，印出這兩個數中較大的數

if else命令

當判斷條件有對與錯時執行動作，Python使用 if else 命令並設定條件對與錯動作，設定方式如下，if、else命令最後須使用:來定義新的程式區塊，新的程式區塊必須縮排，請注意：新的程式區塊必須縮排，否則會發生錯誤。

if (2>1): print('Yes') else: print('No')

if elif else命令

當判斷有許多條件時，第一個為if條件，其他條件則依序為elif條件，else用於設定所有條件都不符合時的動作。

練習：請設定變數 school 值為 南臺科技大學，並利用輸入input()函數讀入1到6的輸入值，當輸入值為1時印出南、當輸入值為2時印出臺，其他值依序印出對應字。 練習：請設定變數 school 值為 南臺科技大學，並利用輸入input()函數讀入1到6的輸入值，如上題，請使用字串位置特性印出對應字。

if命令簡單表示

如果條件僅包含單一執行動作，則可使用下列if命令簡單表示：

if 條件: 條件對動作 a=1 if a>0 :print('yes') 條件對動作 if 條件 else 條件錯動作 print('yes') if a>3 else print('no')

多條件(and、or、not)

當條件超過1個以上，條件與條件之間必須使用and(且)、or(或)來串連這些條件，not用於設定否定。

練習：比較變數a值是否在1到10之間，如果是印出在1到10之間，如果不是印出不在1到10之間 a=15 if a>=1 and a<=10: print('在1到10之間') else: print('不在1到10之間')
